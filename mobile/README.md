# Happiness Mobile App (React Native)

React Native mobile application for iOS and Android.

## 실행 방법

```bash
cd happiness-app/mobile
npm install
npm start
```

Expo가 설치되어 있으면 iOS 시뮬레이터, Android 에뮬레이터 또는 실제 기기에서 앱을 실행할 수 있습니다.

## 기능

- 첫 화면: 사진 갤러리
- 목록 / 상세 페이지: 비로그인 접근 가능
- 사진 등록, 수정, 삭제
- 좋아요, 찜 기능

## API 연동

백엔드 서버는 `http://localhost:8080/api` 또는 Android 에뮬레이터의 경우 `http://10.0.2.2:8080/api`를 사용합니다.

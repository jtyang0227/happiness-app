## 1. 사진 등록 페이지 (PhotoFormPage) 고도화

### 1-1. 현황 & 문제

| 항목 | 현재 | 문제 |
|------|------|------|
| 레이아웃 | 단일 컬럼, 스크롤 | 이미지 프리뷰와 메타데이터 편집이 분리돼 피로감 |
| 이미지 입력 | 파일 업로드 / URL 탭 | 파일 선택 후 즉시 보정 화면으로 이동 없음 |
| 보정 패널 | ImageAdjustmentPanel (긴 스크롤) | 항목이 너무 많아 초보 작가에게 진입 장벽 |
| AI 자동태그 | 버튼 1개 | 추천 태그를 선택/해제하는 UI 없음 |
| 저장 피드백 | alert | 토스트로 대체 필요 |
| 비율 선택 | 5개 버튼 | 선택 후 프리뷰 비율이 변하지 않음 |

### 1-2. 유저 스토리

```
As a 사진작가,
  나는 사진을 올린 직후 보정 패널이 사진 옆에 나타나서
  화면 이탈 없이 보정 → 메타데이터 → 등록을 완결하고 싶다.

As a 입문 작가,
  나는 복잡한 색상 보정 대신 "프리셋 1클릭" 으로 빠르게 분위기를 정하고 싶다.

As a 모든 작가,
  나는 AI가 제안한 태그 중 원하는 것만 골라서 등록하고 싶다.
```

### 1-3. 새 레이아웃 — 2-패널 스튜디오 뷰

```
┌──────────────────────────────────────────────────────────────────┐
│  ← 뒤로   사진 등록 / 수정                           [임시저장] [등록하기]  │
├─────────────────────────────┬────────────────────────────────────┤
│  LEFT — 이미지 프리뷰 (60%)  │  RIGHT — 메타데이터 패널 (40%)      │
│                             │                                    │
│  ┌─────────────────────┐    │  제목 *         [_________________]│
│  │                     │    │  설명           [_________________]│
│  │   사진 프리뷰         │    │                 [_________________]│
│  │   (선택 비율 유지)    │    │  무드 색상       ● ● ● ● ● ● ● ●  │
│  │                     │    │  비율 선택       [16:9][4:3][1:1] │
│  └─────────────────────┘    │  그리드 너비      [──────●──────]  │
│                             │                                    │
│  [⚡ 빠른 프리셋] [🎨 상세 보정] [✏️ 텍스트]      │  ─────────────────────── │
│                             │  AI 추천 태그    [#웨딩] [#인물]   │
│  ← 보정 탭 선택 시           │                  [#서울] [#스냅]   │
│     해당 패널 아코디언 열림    │  [✨ AI 태그 생성]                  │
│                             │                                    │
│                             │  위치(선택)      [________________]│
│                             │  EXIF           [카메라·조리개·ISO] │
└─────────────────────────────┴────────────────────────────────────┘
모바일: 세로 스택 — 프리뷰 → 빠른 보정 탭 → 메타데이터 폼
```

### 1-4. 보정 3단계 전략

| 탭 | 대상 | 포함 항목 |
|----|------|----------|
| ⚡ 빠른 프리셋 | 입문 작가 | 8종 내장 스타일 카드 (Fuji Velvia·Portra 등) |
| 🎨 상세 보정 | 전문 작가 | 현재 ImageAdjustmentPanel 아코디언 전체 |
| ✏️ 텍스트 | 워터마크 필요 작가 | OverlayPanel — 텍스트+워터마크 |

### 1-5. AI 태그 선택 UI

```
[✨ AI 태그 생성] 클릭 후:

추천 태그 (8개)
  [✓ #웨딩]  [✓ #인물]  [  #서울]  [✓ #스냅]
  [  #야외]  [✓ #감성]  [  #봄]    [  #커플]

  → 체크된 태그만 저장 (최대 10개)
  + [직접 입력 +]
```

### 1-6. 드래프트 자동저장

```
저장 주기: 입력 중 3초 debounce
저장 위치: localStorage key = 'photo_draft_{memberId}'
UI: 상단바에 "• 임시저장됨 13:42" 회색 텍스트
복원: 페이지 진입 시 드래프트 감지 → "이전 작업을 이어하시겠습니까?" 토스트
```

### 1-7. 수용 기준

- [ ] 이미지 선택 즉시 왼쪽 프리뷰 영역에 표시, 선택 비율로 크롭 가이드 오버레이
- [ ] 비율 버튼 클릭 시 프리뷰가 해당 비율 aspect-ratio로 즉시 변경
- [ ] 빠른 프리셋 클릭 시 프리뷰에 실시간 반영 (Canvas 적용)
- [ ] AI 태그 생성 후 체크박스로 선택/해제 가능
- [ ] 3초 debounce 드래프트 자동저장 (localStorage)
- [ ] 등록 성공 시 alert 대신 Toast + 사진 상세 페이지로 자동 이동
- [ ] 모바일에서 하단 sticky 등록 버튼

## 3. 갤러리 페이지 (GalleryPage) 고도화

### 3-1. 현황 & 문제

| 항목 | 현재 | 문제 |
|------|------|------|
| 뷰 모드 | masonry / list | 뷰 전환 시 스크롤 위치 초기화 |
| 필터 | 정렬 5종만 | 무드 필터, 비율 필터, 태그 필터 없음 |
| 무한 스크롤 | ❌ (전체 로딩) | 사진이 많으면 버벅임 |
| 그리드 설정 | columns: 4 고정 | 사용자 컬럼 수 조절 불가 |
| 선택 모드 | ❌ | 여러 사진 선택 후 일괄 삭제 없음 |
| 검색 | ❌ | 내 갤러리 내 검색 없음 |

### 3-2. 새 갤러리 툴바 설계

```
┌──────────────────────────────────────────────────────────────────┐
│  🔍 내 갤러리 검색...   [무드▾] [비율▾] [정렬▾]   □선택   ⊞≡  N장 │
└──────────────────────────────────────────────────────────────────┘

무드 드롭다운:
  전체 | ● 따뜻함 | ● 시원함 | ● 드라마틱 | ● 로맨틱 | ...

비율 드롭다운:
  전체 | 16:9 | 4:3 | 1:1 | 3:4 | 2:3

정렬 드롭다운:
  최신순 | 오래된 순 | 좋아요 순 | 저장 순 | 표시 순서

⊞≡: masonry / list / grid-2 / grid-3 (4종 뷰)
```

### 3-3. 4종 뷰 모드

| 뷰 | 설명 | 사용 케이스 |
|----|------|------------|
| masonry | CSS columns 4컬럼, 모바일 2컬럼 (현재) | 감성 포트폴리오 느낌 |
| list | 72px 썸네일 + 제목/통계 행 (현재) | 빠른 관리 |
| grid-2 | 2컬럼 정사각 그리드 | 인스타 스타일 |
| grid-3 | 3컬럼 정사각 그리드 | 많은 사진 한눈에 |

### 3-4. 무한 스크롤 (useIntersectionObserver)

```javascript
// hooks/useInfiniteGallery.js
export function useInfiniteGallery(params, pageSize = 20) {
  const [items, setItems] = useState([]);
  const [page, setPage]   = useState(0);
  const [hasMore, setHasMore] = useState(true);
  const [loading, setLoading] = useState(false);
  const sentinelRef = useRef(null);

  useEffect(() => {
    const io = new IntersectionObserver(([entry]) => {
      if (entry.isIntersecting && hasMore && !loading) {
        setPage(p => p + 1);
      }
    }, { threshold: 0.1 });
    if (sentinelRef.current) io.observe(sentinelRef.current);
    return () => io.disconnect();
  }, [hasMore, loading]);

  useEffect(() => {
    setItems([]); setPage(0); setHasMore(true);
  }, [JSON.stringify(params)]);

  // page 변경 시 데이터 로드
  useEffect(() => { loadPage(page); }, [page]);

  return { items, loading, hasMore, sentinelRef };
}
```

### 3-5. 선택 모드 (Multi-select)

```
[□ 선택] 버튼 클릭 시 선택 모드 진입:
  - 각 사진 카드 좌상단에 체크박스 표시
  - 선택된 사진 테두리 primary 색
  - 하단 sticky 바 등장: "3장 선택됨  [시리즈에 추가] [일괄 삭제]"
```

### 3-6. 갤러리 내 검색

```
검색어 입력 시 클라이언트 필터 (title, description, tags, colorMood 매칭)
실시간 결과 (debounce 200ms)
"검색 결과 없음" EmptyState 표시
```

### 3-7. 수용 기준

- [ ] 무드 드롭다운 필터 (무드 선택 시 해당 무드 사진만 표시)

## 4. claude.ai 아티팩트 프롬프트 — 사진 등록 2-패널 레이아웃

```
[시스템 컨텍스트]

기술 스택: React 18 SPA, React Router v6, inline style (CSS-in-JS 없음)
아이콘: 이모지 또는 유니코드 기호 사용 (외부 아이콘 라이브러리 없음)

  글래스 패널:  backdrop-filter: blur(20px), background: rgba(255,255,255,0.72)
  글래스 보더:  1px solid rgba(255,255,255,0.5)

- 한국어 UI 텍스트

사진 등록 페이지를 2-패널 스튜디오 레이아웃으로 만들어줘.

  1. 이미지 업로드 영역 (이미지 없을 때):

     - 하단에 "파일 업로드" / "URL 입력" 탭
```

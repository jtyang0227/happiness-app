## 컴포넌트 프롬프트

React 18 + inline style로 사진작가-모델 약속 커뮤니케이션 공간을 만들어줘.
배경 #090909 다크, Cosmos 다크 에디토리얼 스타일.

핵심 3개 화면:
1. MeetsPage: 탭 필터 + 약속 카드 리스트 (Cosmos 언더라인 탭)
2. MeetDetailPage: 날짜조율 달력 + 지도 + 채팅 (3섹션 레이아웃)
3. MeetRequestModal: 3단계 위저드 (날짜→장소→메시지)

달력: 순수 JS Date API, 복수 날짜 선택, 겹치는 날짜 초록색 강조.
지도: Kakao Maps SDK 동적 로드, 키워드 검색 후 마커 표시, 미설정 시 텍스트 입력 fallback.
채팅: 30초 폴링, 내 메시지 오른쪽, 상대방 왼쪽.

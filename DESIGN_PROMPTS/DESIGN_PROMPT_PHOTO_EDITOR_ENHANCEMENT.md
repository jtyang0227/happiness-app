## 프롬프트 A — ImageAdjustmentPanel v2 (Camera Calibration 추가)

```
위 시스템 컨텍스트를 기반으로 React 18 ImageAdjustmentPanel 컴포넌트를 디자인해줘.

이 컴포넌트는 이미지 보정 에디터의 우측 패널이야.
에디터 패널 배경(#0c0c18)에 흰/밝은 텍스트 톤으로 설계해.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

레이아웃
- 고정 너비 300px, 전체 높이 스크롤 가능
- 각 섹션은 아코디언 (클릭 열림/닫힘)
- 변경이 있는 섹션 헤더에는 파란 ● 뱃지 표시

섹션 순서 (Lightroom 표준 순서):
1. 기본 보정 (Basic)
2. 톤 커브 (Tone Curve)  
3. 색상 혼합 (Color Mixer / HSL)
4. 색 보정 (Color Grading)
5. 디테일 (Detail) — Sharpening + Noise Reduction 통합
6. 📷 카메라 캘리브레이션 (Camera Calibration) ← NEW
7. 효과 (Effects) — Texture·Clarity·Dehaze·Vignette·Grain

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[6] 카메라 캘리브레이션 섹션 상세 디자인:

섹션 헤더:
  📷 카메라 캘리브레이션              ●(변경 시)    ∨

섹션 열렸을 때:
  ┌─────────────────────────────────────────┐
  │  Red Primary                            │
  │    Hue         [━━━━━━●━━━━━━━━━]  +20 │   ← 붉은 그라디언트 트랙
  │    Saturation  [━━━━━━━━━━━●━━━━]  +10 │
  │                                         │
  │  Green Primary                          │
  │    Hue         [━━━━━●━━━━━━━━━━]  -15 │   ← 초록 그라디언트 트랙
  │    Saturation  [━━━━━━━━━━●━━━━━]  +10 │
  │                                         │
  │  Blue Primary                           │
  │    Hue         [●━━━━━━━━━━━━━━━]  -35 │   ← 파란 그라디언트 트랙
  │    Saturation  [━━━━━━━━━━━━━●━━]  +35 │
  │                                         │
  │              [초기화]                   │
  └─────────────────────────────────────────┘

슬라이더 트랙 그라디언트:
  - Red Hue:        left=#2ea44f → center=#e53e3e → right=#ff8000
  - Red Sat:        left=#ccc → right=#e53e3e
  - Green Hue:      left=#0288d1 → center=#43a047 → right=#ffb300
  - Green Sat:      left=#ccc → right=#43a047
  - Blue Hue:       left=#5b6ef5 → center=#0288d1 → right=#43a047
  - Blue Sat:       left=#ccc → right=#5b6ef5

색 보정(Color Grading) 섹션 하단에 Balance 슬라이더 추가:
  Balance  [━━━━━━━━●━━━━━━━━━━━━━━━]  -10
  (왼쪽 = 섀도 확장, 오른쪽 = 하이라이트 확장, 중앙 텍스트로 표시)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

공통 슬라이더 디자인:
- 트랙 높이 4px, 핸들 크기 14px 원형, 흰색 테두리
- 현재 값이 0이 아니면 파란 색상으로 구분
- 슬라이더 오른쪽에 숫자 값 (폭 36px 고정, 오른쪽 정렬)
- 더블클릭 시 0으로 리셋 (또는 기본값)

섹션 헤더 디자인:
- 배경 #1a1a3a, hover 시 #22225a
- 좌측 아이콘 + 텍스트, 우측 ∨/∧ 토글 + 변경 뱃지
- 변경 뱃지: 파란 ● 6px 원형
```
